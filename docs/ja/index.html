特長

レスポンシブレイアウト

Bootstrap 4のレスポンシブレイアウトに対応しているため、PCでもモバイルデバイスでも、コンテンツが最適化されて表示されます。

カスタムメニュー

ヘッダーにメニューを追加することができます。メニューの項目は好きなページやURLを設定可能。階層化したプルダウンメニューにも対応しています。

カスタムロゴ

好きな画像を選んでサイトのロゴを設置することができます。 個性的なロゴマークであなただけのブランドイメージを演出しましょう。

カスタムヘッダー

好きな画像やYoutube動画などを選んでヘッダーを設置することができます。サイトのアクセントとして、最新の活動報告などを掲載してみてはいかがでしょう？

ウィジェット

フッターに各種リンクやパーツを追加できます。検索ボックスやカレンダー、ギャラリーなど個性豊かなウィジェットでホームページを彩りましょう。

テーマカスタマイザー

サイト上の項目やスタイルは「テーマカスタマイザー」を使って編集できます。編集結果をリアルタイムに確認できるので操作はとっても簡単です。

テーマ名の由来

「tsumugi」というテーマ名は、シンガーソングライター・初音さんの「紡ぎ〜あまねく想い〜」という曲のタイトルから名付けさせていただきました。
歌のコンセプトと同じように、「tsumugi」も言葉を『紡ぐ』ことの大切さを込めて制作しています。

オリジナルグッズ

「tsumugi」のオリジナルグッズを販売しています。グッズの収益はテーマの開発費に充てさせていただきますので、気に入った方はぜひお買い求めいただけると嬉しいです。
