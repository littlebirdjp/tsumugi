「tsumugi」は、文章が読みやすく、
想いを伝えやすい、
誰にでも使えるシンプルなWordPressテーマです。

無料ダウンロードデモサイト

コンセプト

モバイルフレンドリー

スマートフォンやタブレットに最適化されているので、モバイル環境でも読みやすいデザインを実現します。

  実際の見え方は「デモサイト」をご覧ください。

ユーザーフレンドリー

複雑な機能やオプションを搭載していないので、初心者の方でも簡単に利用することができます。

  利用できる機能については「特長」をご覧ください。

サステイナビリティー

拡張しやすい作りになっているため、いつでも最新のフレームワークにアップデートすることができます。（Bootstrap v4.0.0対応）

  テーマの編集方法については「カスタマイズ」をご覧ください。

特長

レスポンシブレイアウト

Bootstrap 4のレスポンシブレイアウトに対応しているため、PCでもモバイルデバイスでも、コンテンツが最適化されて表示されます。

カスタムメニュー

ヘッダーにメニューを追加することができます。項目の編集は、管理画面の「外観」→「メニュー」から簡単に行えます。

カスタムロゴ

好きな画像を選んでサイトのロゴを設置することができます。
※「tsumugi」のロゴを使いたい場合は、テーマディレクトリ内のimgフォルダに同梱されているファイルをご使用ください。

カスタムヘッダー

好きな画像を選んでヘッダーに設置することができます。初期設定ではオリジナルの写真が表示されます。
※ヘッダー画像は削除することも可能です。

ウィジェット

フッターにカテゴリーやタグ、アーカイブ等のリンクやパーツを設置できます。項目の編集は、管理画面の「外観」→「ウィジェット」から簡単に行えます。

テーマカスタマイザー

その他、様々な項目やスタイルを「テーマカスタマイザー」を使って編集できます。テーマの編集は「外観」→「カスタマイズ」から簡単に行えます。

カスタマイズ

開発者・デザイナーの方へ

「tsumugi」を本格的にカスタマイズするには、ダウンロード版ではなく、開発用のファイルも含んだGitHubバージョンをご利用ください。
こちらのリポジトリに、開発ツールの使用方法や、制作過程などが記載されています。また、本テーマはオープンソースで公開されているので、フォークやプルリクも歓迎しています。

テーマ名の由来

「tsumugi」というテーマ名は、シンガーソングライター・初音さんの「紡ぎ〜あまねく想い〜」という曲のタイトルから名付けさせていただきました。
歌のコンセプトと同じように、「tsumugi」も言葉を『紡ぐ』ことの大切さを込めて制作しています。

オリジナルグッズ

「tsumugi」のオリジナルグッズを販売しています。グッズの収益はテーマの開発費に充てさせていただきますので、気に入った方はぜひお買い求めいただけると嬉しいです。
